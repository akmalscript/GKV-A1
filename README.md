### Mengurangi Sensitivitas Gerakan Maju/Mundur
```c
// Sebelumnya
x = x + i*(lx)*0.1;
z = z + i*(lz)*0.1;

// Setelah dikurangi
x = x + i*(lx)*0.03;
z = z + i*(lz)*0.03;
```

### Mengurangi Sensitivitas Belok Kanan/Kiri
```c
// Sebelumnya
case GLUT_KEY_LEFT : deltaAngle = -0.01f;break;

// Setelah dikurangi
```
